# hrm


# you should have nodeJS installed in your system


first make a clone of git repository using "git clone "repo link" with terminal <br>
then change the directory to the repository you just cloned in your system <br>
then on your 1st terminal change directory to server using "cd server" and then use "npm install" then use "npm start" in terminal to start the server<br>
then on your 2nd terminal change directory to client using "cd client" and then use "npm install" then "npm start" in terminal to start client (front-end),
